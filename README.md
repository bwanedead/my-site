# Signal/Frame

A personal work showcase shell built in an aesthetic-first first pass: dark, cinematic, modular, and ready to hold project writeups later.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Three.js
- React Three Fiber
- Drei
- TypeScript

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Structure

- `app/`: App Router routes for home, work index, project detail, and contact
- `components/sections/`: landing page modules
- `components/ui/`: reusable panels and motion primitives
- `components/canvas/`: atmospheric Three.js layer for the hero
- `lib/site-content.ts`: placeholder project data
